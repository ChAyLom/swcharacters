# Star Wars Characters

Сайт для просмотра информации о персоонажах вселенной Звездных Войн.

## Сборка и запуск

Установка зависимостей:

```sh
npm ci
```

Запуск локального сервера для разработки:

```sh
npm start
```

Сборка production билда:

```sh
npm run build-prod
```

Сборка development билда:

```sh
npm run build-dev
```
